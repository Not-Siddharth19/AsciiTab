Install

Get AsciiTab running in your browser.

  1. Go to addons.mozilla.org
  2. Search for "AsciiTab"
  3. Click Add to Firefox
  4. Click Add when prompted
  5. Open a new tab to see AsciiTab
  1. Download the latest release from the theme's GitHub repo
  2. Extract the ZIP file to a folder
  3. Open chrome://extensions
  4. Enable Developer mode (top right)
  5. Click Load unpacked
  6. Select the extracted folder
  7. Open a new tab to see AsciiTab

Is AsciiTab free?

Yes! Completely free and open source under MPL 2.0. No premium features, no paywalls.

Where is my data stored?

All data stays in your browser's local storage. Nothing is sent to external servers.

Does it work on mobile?

AsciiTab is designed for desktop browsers. Mobile support is not available yet.

How do I add custom commands?

Use @s to open settings, then navigate to Commands to add custom shortcuts.
