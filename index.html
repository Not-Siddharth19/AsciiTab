About AsciiTab

AsciiTab aims for a minimalist, keyboard-driven start page experience. Built for developers who value speed, simplicity, and a terminal-like aesthetic.

Replace your browser's new tab with a command center. No clutter, no bloat — just the tools you need, accessible from your fingertips.

Minimalism Manager

Three levels of focus. Choose your intensity.

Mode 1: Balanced

ASCII Art + Clock + Date + Search. Sidebar hidden.

Mode 2: Concentrated

ASCII Art + Search Bar. Time hidden.

Mode 3: Zen

ASCII Art only. Pure aesthetics.

Use @min to open the Minimalium Manager

Install Now

Get started with AsciiTab in just a few steps.

  1. Download the latest .zip release from GitHub
  2. Extract the ZIP file to a folder
  3. Open chrome://extensions
  4. Enable Developer mode (top right toggle)
  5. Click Load unpacked
  6. Select the extracted folder
  7. Open a new tab to see AsciiTab
